🛴 Scoot Multi-Page Frontend
A fully responsive, multi-page frontend application built with React and Tailwind CSS. This project serves as a showcase of advanced frontend layouting techniques and modern web design principles.

Key highlights include:

1. Pixel-Perfect Design Translation: Meticulous attention to detail bridging the gap between static Figma designs and fluid web interfaces.

2. Advanced Responsive Layouts: Complex handling of flex and grid systems to ensure flawless presentation across mobile, tablet, and desktop viewports.

3. Dynamic Asset Positioning: Intricate absolute positioning and scaling of SVG patterns and decorative background elements that respond dynamically to screen size.

4. Scalable Component Architecture: Highly reusable UI components (like FeatureRow and Button) designed to accept dynamic props and alignment overrides.
